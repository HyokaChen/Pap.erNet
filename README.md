# Pap.erNet
参考MacOS上面的[pap.er](https://paper.photos)，采用[Avalonia](https://github.com/AvaloniaUI/Avalonia)跨平台技术实现的Windows /Liunx 下的壁纸软件。

## 开发
```bash
git clone https://github.com/HyokaChen/Pap.erNet.git
cd Pap.erNet/
./build-win.bat 或者 ./build-linux.bat
```
